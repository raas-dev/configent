# Open Computer Use

MCP server to automate computer use.

See [open-codex-computer-use](https://github.com/iFurySt/open-codex-computer-use)

## Setup

Install:

```sh
npm i -g open-computer-use
```

On macOS, run it once and grant Accessibility and Screen Recording:

```sh
open-computer-use
```

Windows and Linux do not need this step.

Check permissions anytime:

```sh
open-computer-use doctor
```
